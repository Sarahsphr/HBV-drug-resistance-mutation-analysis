# HBV Drug Resistance Mutation Analysis

Python-based bioinformatics workflows for identifying resistance-associated mutations in Hepatitis B Virus (HBV) genomic datasets.

## Project Status

Ongoing manuscript-stage bioinformatics workflow. This repository contains analysis scripts and computational methods only.

## Included Analyses

* Core protein mutation analysis
* Surface protein mutation analysis
* Reverse transcriptase mutation analysis
* X region mutation analysis

## Features

* FASTA alignment processing
* Mutation detection
* Genotype-specific mutation counting
* CSV summary generation
* Automated workflow scripting

## Technologies

* Python
* Biopython
* CSV processing
* Regular expressions
